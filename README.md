# Healthcare_provider_dashboard
This project is an Interactive Healthcare Provider Dashboard developed using Power BI to analyze hospital billing and healthcare service data. The dashboard helps visualize important financial and operational metrics, enabling better understanding of healthcare costs and service distribution.

📊 Healthcare Provider Dashboard – Power BI Project
📌 Project Overview

This project is an Interactive Healthcare Provider Dashboard built using Power BI to analyze healthcare billing data and provide insights into hospital financial performance, treatment costs, and department-level analysis.

The dashboard helps visualize key metrics such as billing amount, medication cost, treatment cost, insurance coverage, and out-of-pocket expenses. It also enables users to explore healthcare data using interactive filters and dynamic visualizations.

🎯 Objectives
The goal of this project is to:
Analyze healthcare billing data
Identify cost distribution across treatments and departments
Understand procedure usage trends
Provide insights for better financial and operational decision-making

🛠 Tools & Technologies Used

Power BI Desktop – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Creating KPIs and calculated measures
Microsoft Excel – Initial data preparation

📊 Dashboard Features
1️⃣ KPI Cards

The dashboard displays key healthcare financial metrics:
Total Billing Amount
Medication Cost
Treatment Cost
Total Insurance Coverage
Out-of-Pocket Expenses
Room Charges
Each KPI also shows average cost metrics per patient visit.

2️⃣ Procedure Analysis

This section analyzes the total billing amount and percentage distribution of medical procedures, including:
X-Ray
CT Scan
MRI
Ultrasound
Blood Test
This helps identify which procedures generate the highest revenue.

3️⃣ Department-wise Billing Analysis

The dashboard shows total billing amount by department, including:
Cardiology
Orthopedics
General Surgery
Neurology
Pediatrics
This helps understand which departments contribute the most to overall hospital revenue.

4️⃣ Diagnosis & Service Type Analysis

This visualization shows billing distribution based on diagnosis and service type:
Diagnosis examples:
Hypertension
Appendicitis
Asthma
Fracture
Migraine
Service types:
Emergency
Inpatient
Outpatient
This helps analyze patient service patterns and treatment types.

5️⃣ Geographic Analysis

A map visualization displays the total billing amount by city, helping identify geographic trends in healthcare service usage.

6️⃣ Interactive Filters

Users can explore the dashboard using dynamic filters such as:
Patient Race
Year / Quarter / Month
City / State
These filters allow deeper analysis of healthcare data.

7️⃣ Dashboard UI Features

This dashboard also includes advanced UI functionality:
🌙 Dark Mode & Light Mode Toggle
🔍 Open / Close Filter Panel
📊 Interactive charts and slicers
📍 Map visualization  
📈 KPI cards for quick insights
